Background column configuration

-bicol/--background-id-column

Column in background metadata file (specified using -bm/--background-metadata) to match input IDs against. Default: sequence_name

-sicol/--sequence-id-column

Column in background metadata file to match with IDs in the sequence file

-bdate/--background-date-column

Column in background metadata file that contains date information. Must be in the format YYYY-MM-DD. Default: sample_date

-bloc/--background-location-column

Column in the background metadata file that contains geographical information. Defualt: country
