Misc options

These are a collection of options that didn't fit into other groups and are mostly to do with the running of civet

--civet-mode

If this is equal to "CLIMB", then UK-specific defaults will be used for some arguments.


Default:
GLOBAL


Usage:
--civet-mode CLIMB or --civet-mode GLOBAL

-r/--reference-sequence

Custom reference genome to map and pad against. Must match the reference the background sequence alignment was generated from.


Default:
WH04.fasta (SARS-CoV-2 reference sequence)


Usage:
--reference-sequence {$SEQUENCE_FILE}

-mem/--max-memory

Indicates the maximum amount of RAM (in GB) to use for tree building.


Default:
8


Usage:
--max-memory {$NUMBER}

--art

Flag to show artwork


Default:
False


Usage:
--art

-t/--threads

Number of threads used in running civet


Default:
1


Usage:
--threads {$NUMBER}

--verbose

Flag to provides more information on the processes being run by printing to screen


Default:
False


Usage:
--verbose

-v/--version

Flag to show the civet version and exit


Default:
False


Usage:
--version

-h/--help

Prints help associated with all of the arguments


Default:
False


Usage:
--help
