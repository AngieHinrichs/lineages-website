Output options

-o/--outdir

Path to output directory. Default: civet_DATESTAMP

-p/--output-prefix

Prefix of output directory and report name. Default: civet

--datestamp

Add datestamp to directory name specified -o/--outdir. Default: False

--overwrite

Flag to overwrite the output directory. Default: False - a number wil be added to the the output directory.

--output-data

Flag to save intermediate data files in the output directory

-temp/--tempdir

Path to temporary directory.

--no-temp

Flag to output all intermediate files, for development/debugging purposes.
