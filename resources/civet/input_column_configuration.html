Input column configuration

-icol/--input-id-column

Column in input metadata file (specified usingn -i/--input-metadata) to match with background database. Default: name

-idisp/--input-display-column

Column containing alternative sequence names for displaying in the report, for example patient IDs

-idate/--input-date-column

Column in input metadata containing date information. Default: sample_date
